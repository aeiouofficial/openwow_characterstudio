# Character Studio v4.2.0 — Unified Workspace UX

This release reorganizes the existing production features into a clearer professional workspace without removing the v4.1 character, Library, scene, timeline, camera, audio, portability, or render systems.

## Highlights

- Five task-focused Machinima layouts: Edit, Animate, Camera, Audio, and Review.
- Collapsible and resizable Production Browser, Properties, stage, timeline, Character, and Materials & Output docks.
- Searchable accordion Properties inspector with persisted states.
- `Ctrl+K` command palette and keyboard layout shortcuts `1` through `5`.
- Main editor panel search plus All/Core/Advanced setting scopes.
- Direct animation preview, duration badges, and loop/hold clip controls.
- Guided first-shot empty states, compact overlay docks, click-away dismissal, accessible separators, and reduced-motion support.
- Refined brushed-metal surfaces, visual hierarchy, controls, borders, spacing, hover/selected states, and responsive behavior.
- Compact top-bar prioritization keeps Open Model, Save Character, Library, and Scene Studio visible without horizontal scrolling.
- Storage-safe Machinima autosave avoids noisy failures when browser persistence is restricted.

All prior v4.1.1 full-character saving and v4.1.0 Machinima Studio Elite workflows remain available.
